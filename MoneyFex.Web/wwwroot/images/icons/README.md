# Icon Assets

This directory should contain SVG icons for the transaction types:

- `bank.svg` - Bank Account Deposit icon
- `mobile-money.svg` - Mobile Money Transfer icon
- `cash-pickup.svg` - Cash Pickup icon
- `kiipay.svg` - KiiBank/KiiPay Wallet icon
- `logo.png` - MoneyFex logo

## Temporary Solution

Until actual icons are added, the application uses Font Awesome icons via CDN:
- Bank: `<i class="fas fa-university"></i>`
- Mobile: `<i class="fas fa-mobile-alt"></i>`
- Cash: `<i class="fas fa-hand-holding-usd"></i>`
- Wallet: `<i class="fas fa-wallet"></i>`

## Icon Requirements

- Format: SVG (preferred) or PNG
- Size: 50x50px for home page, 60x60px for service cards
- Style: Match MoneyFex brand colors (#02b6ff primary)
- Background: Transparent
